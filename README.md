# Proyecto Valleny

## Descripción
Página web para local de coctelería y eventos en vivo

## Instalación
1. Clonar repositorio: `git clone https://github.com/MarioAFdev/proyecto-valleny.git`
2. Abrir `index.html` en el navegador

## Características
- Filtro por fecha
- Filtro por género musical
- Diseño responsive
